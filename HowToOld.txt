I have a python script based on the imx500 pose estimation script demo
python3 ~/AICAM/imx500-models/picamera2/examples/imx500/pose_sword.py --unity-ip 144.39.120.28
You must tell it your IP address so it can send UDP packets with this JSON
{
    "shoulders": {
        "left":  kp(IDX_L_SHOULDER),
        "right": kp(IDX_R_SHOULDER)
    },
    "hips": {
        "left":  kp(IDX_L_HIP),
        "right": kp(IDX_R_HIP)
    },
    "sword": {
        "tip":               tip,
        "handle":            handle,
        "direction":         [round(dx, 4), round(dy, 4)],
        "length":            round(length, 4),
        "angle_deg":         round(angle, 2),
        "angle_from_vertical": round(angle_from_vertical, 2),
        "detected":          sword_detected
    },
    "t": round(time.time(), 4)
}

How I set this up:
Set up the Pi. Originally this was at HackUSU so I set things up to run through Eduroam wifi and you have to bypass the firewalls on whatever computer is receiving the packets.
We had a computer running the game in Unity that receives the info from a ras pi.

I followed joinlater github repo to connect pi to Eduroam at USU:
sudo apt install python3-pip python3-venv -y
mkdir joinlater
cd joinlater
python3 -m venv joinlater
source joinlater/bin/activate
pip install https://github.com/Xeyler/joinlater/releases/download/v0.2.1/joinlater-0.2.1.tar.gz
joinlater
deactivate


Actually setting up the Pi models and imx500 camera
mkdir AICAM
cd AICAM

sudo apt install imx500-all

Clone the imx500 models:
git clone https://github.com/raspberrypi/imx500-models
cd imx500-models
./download_models.sh


Also clone the picamera2 repo: (I think this is the only repo I ended up using)
git clone https://github.com/raspberrypi/picamera2.git

Get open CV:
sudo apt install python3-opencv

And then add a proxy for where the folders were at so they could be found by the script.
You can see from the folder structure where the demos are that you can run with the imx500 Ras Pi AI camera.
There are .rpk files in just the imx500 repo that you could run, but I cloned the picamera2 repo and that one had the .py demos ands thats what I use.
sudo ln -s ~/AICAM/imx500-models /usr/share/imx500-models
sudo mkdir -p /usr/share/imx500-models
sudo ln -s ~/AICAM/imx500-models/imx500_network_ssd_mobilenetv2_fpnlite_320x320_pp.rpk /usr/share/imx500-models/imx500_network_ssd_mobilenetv2_fpnlite_320x320_pp.rpk
cd ../..
sudo mkdir -p /usr/share/imx500-models
sudo ln -s ~/AICAM/imx500-models/imx500_network_ssd_mobilenetv2_fpnlite_320x320_pp.rpk /usr/share/imx500-models/imx500_network_ssd_mobilenetv2_fpnlite_320x320_pp.rpk
